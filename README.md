Простое и элегантное веб-приложение для управления списком задач с сохранением данных в localStorage. Проект написан на чистом JavaScript (ES6+), HTML и CSS. Реализованы основные CRUD-операции, поиск, фильтрация и анимация удаления.

✨ Функциональные возможности
➕ Добавление новой задачи

✅ Отметка о выполнении (чекбокс)

❌ Удаление отдельной задачи

🗑️ Удаление всех задач (с подтверждением)

🔍 Поиск задач по заголовку (фильтрация в реальном времени)

💾 Автоматическое сохранение всех задач в localStorage

💬 Информативные сообщения: «There are no tasks yet» или «Tasks not found»

🎨 Анимация при удалении (плавное исчезновение и сдвиг соседних элементов)

🛠️ Технологии
HTML5

CSS3 (Flexbox, Grid, кастомные свойства, BEM-именование)

JavaScript (ES6+ классы, события, работа с DOM, localStorage)

Нормализация стилей с помощью normalize.css

[![HTML5](https://img.shields.io/badge/markup-HTML5-E34F26?logo=html5)](https://developer.mozilla.org/en-US/docs/Web/HTML)
[![localStorage](https://img.shields.io/badge/storage-localStorage-yellowgreen)](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage)

> Простое и элегантное веб-приложение для управления списком задач.  
> Все данные сохраняются в `localStorage` — после перезагрузки страницы ваши задачи останутся на месте.

---

## ✨ Функциональные возможности

| Действие | Описание |
|----------|----------|
| ➕ **Добавление** | Новая задача появляется в списке после нажатия кнопки или клавиши `Enter`. |
| ✅ **Отметка о выполнении** | Чекбокс — визуально зачёркивает текст и сохраняет статус. |
| ❌ **Удаление одной задачи** | Кнопка «🗑️» рядом с каждой задачей. |
| 🗑️ **Удаление всех задач** | Кнопка «Clear all» с подтверждением (alert / confirm). |
| 🔍 **Поиск** | Фильтрация задач по заголовку в реальном времени. |
| 💾 **Автосохранение** | Любое изменение (добавление, удаление, переключение статуса) мгновенно сохраняется в `localStorage`. |
| 💬 **Информативные сообщения** | Показываются если задач нет или результаты поиска пусты. |
| 🎨 **Анимация удаления** | Плавное исчезновение и сдвиг соседних элементов (CSS transition). |

---

## 🛠️ Технологии

- **HTML5** — семантическая разметка, доступность.
- **CSS3** — Flexbox / Grid, кастомные свойства (CSS variables), плавные анимации, BEM-именование классов.
- **JavaScript (ES6+)** — классы (ES6 Classes), работа с DOM, события, `localStorage`, методы массивов (`filter`, `map`, `forEach`).
- **Normalize.css** — обеспечивает одинаковое отображение во всех браузерах.
